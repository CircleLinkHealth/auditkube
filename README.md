# auditkube

Secure Images to use with [Kubernetes
Kops](https://github.com/kubernetes/kops) to help create images which help
with PCI/HIPAA Compliance.

# Features

 - [X] Encrypted Root Volume
 - [X] [OSSEC](https://ossec.github.io/)
